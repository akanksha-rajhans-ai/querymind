# QueryMind v0.1.0 MVP

Initial MVP for QueryMind, a schema-aware natural language to SQL demo.

## Included

- Local SQLite ecommerce demo database
- Schema introspection over tables, columns, row counts, and foreign keys
- Lightweight schema retrieval for grounding
- Deterministic text-to-SQL templates for common analytics questions
- Read-only SQL guardrails
- Browser UI with SQL, schema grounding, and results
- Evaluation harness with 6 passing smoke cases
- Architecture diagram and launch copy
